# 開発環境

Ruby 2.3.3
Ruby on Rails 5.1.4

# 本番環境

Herokuを使用し、Github上のソースコードにコミットが確認された段階で自動的に最新版をデプロイするようにしてあります。
そのため、長期的に新機能の開発をしている際には挙動が不安定になることもあるかと思います。
あらかじめご了承ください。
